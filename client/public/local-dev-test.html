Set up your local development environment with mock data

1. Enable Mock Data Mode

Add ?mock=true to your dashboard URL or enable it programmatically.

Example URLs:

  • http://localhost:3000/employer-dashboard?mock=true
  • http://localhost:3000/employer-dashboard/applications?mock=true
  • http://localhost:3000/employer-dashboard/manage-jobs?mock=true

2. Start Development Server

Run your Next.js development server:

npm run dev

or

yarn dev

3. Navigate to Dashboard

Open your browser and go to:

http://localhost:3000/employer-dashboard?mock=true

You should see the dashboard with mock data!
